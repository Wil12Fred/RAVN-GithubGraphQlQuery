# Installación

Clonar este repositorio.

## Update
 En actualizar variable AUTH_APOLLO en: /app/src/main/java/com/example/ravn/utilities/Constants.kt
 
 AUTH_APOLLO = ``OAUTH ACCES TOKEN``
  
 obtener token en https://github.com/settings/tokens
